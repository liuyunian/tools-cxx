# tools-cxx
C++工具库 -- 造轮子使人快乐

## 编译
### 环境 & 依赖  
1. 测试OS：ubuntu18.04和MacOS 10.15  
2. 测试编译器：gcc version 7.3.0

### 构建  
1. 构建：make  
2. 安装：make install   
3. 卸载：make uninstall  
4. 清理：make clean

## 示例  
示例程序位于examples目录下  
1. 构建：make  
2. 执行：构建生成的可执行程序位于examples/build/xxx目录下  
3. 清理：make clean
